### 스택/큐(Stack/Queue)
https://mygumi.tistory.com/357 <br>
양쪽으로 접근하는 리스트와 달리 확실히 입구/출구를 정한 형태다. 예컨대 스택은 컴퓨터 코드 실행 구조가 있고 큐는 네트워크 상태 확인이 있다.(링크 확인) 

Stack: LIFO(Last Input First Output)<br>
push / pop

Queue: FIFO(first Input First Output)<br>
